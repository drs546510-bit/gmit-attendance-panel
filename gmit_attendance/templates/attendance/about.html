⚙️ What This App Does
🏛️
7 Branches, 8 Semesters

EE, CS, AI, EC, ME, CE, and CSD, each tracked across all 8 semesters independently.

📥
Bulk Excel Import

Admins can import an entire class list in one upload instead of adding students one by one.

✅
One-Click Attendance

Teachers mark Present/Absent instantly — no save button, no page reloads.

🔐
Role-Based Access

Admins approve teacher accounts before they can mark attendance; students get view-only access.

📊
Live Charts

Branch-wise and class-wise attendance trends, visualized automatically.

🗄️
MySQL Powered

Built on Django + MySQL for reliable, structured record-keeping.
